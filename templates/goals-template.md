# Goals Example

Example of how to define goals in `GOALS.md` so AI can align task priorities and recommendations.

---

## How It Works

1. **Define 3-5 quarterly goals** — Too many = unfocused
2. **Update quarterly** or when priorities shift
3. **AI references this during**:
   - Weekly reviews (goal progress tracking)
   - Daily planning (priority alignment)
   - Backlog processing (strategic task categorization)

---

## Example Content

```markdown
# Goals

## Current Quarter Goals

### Goal 1: Improve Mobile Retention

**Description**: Reduce Day-30 churn on mobile by 15%

**Why it matters**: Mobile is 60% of new signups but retention lags desktop by 20pp

**Success criteria**:
- [ ] Identify top 3 drop-off points via funnel analysis
- [ ] Ship onboarding improvements (target: +10% activation)
- [ ] Launch push notification re-engagement flow

**Target date**: 2026-03-31

---

### Goal 2: Launch Enterprise SSO

**Description**: Ship SAML-based SSO for enterprise tier

**Why it matters**: Blocking $2M pipeline — 3 enterprise deals waiting on this

**Success criteria**:
- [ ] SAML integration tested with Okta and Azure AD
- [ ] Admin console for SSO configuration
- [ ] Migration path for existing enterprise accounts

**Target date**: 2026-02-28

---

### Goal 3: Reduce Support Ticket Volume

**Description**: Cut weekly support tickets by 20% through self-serve improvements

**Why it matters**: Support team is at capacity; hiring freeze means we need to scale through product

**Success criteria**:
- [ ] Launch in-app help center with top 20 FAQs
- [ ] Add status page for real-time incident updates
- [ ] Improve error messages on top 5 failure scenarios

**Target date**: 2026-03-15
```

---

## Tips

- Make goals specific and measurable — vague goals get vague results
- Include "why it matters" — helps AI prioritize trade-offs
- Archive completed goals at the bottom of the file for reference
- Link related tasks back to goals in their Context section

---

**Remember**: Goals drive everything. AI uses these to keep your work aligned with what matters most.
