# Goals — Q2 2026

## Goal 1: Complete Core Transaction Infrastructure

**Description**: Complete Interac and OBV integrations; eliminate remaining payment processing reliability issues

**Why it matters**: Transaction reliability is table stakes — every merchant outage is lost revenue and eroded trust

**Success criteria**:
- [ ] Complete Interac Core transaction migration
- [ ] Resolve OBV verification failures in production
- [ ] Achieve 99.9% payment success rate across all rails

**Target date**: 2026-06-30

## Goal 2: Drive Merchant Growth to 500 Active Merchants

**Description**: Grow active merchant base from current state to 500 by end of Q2

**Why it matters**: Revenue milestone; proves product-market fit at scale before Series A

**Success criteria**:
- [ ] Reduce time-to-first-transaction from sign-up to under 48 hours
- [ ] Achieve 85% 30-day merchant activation rate
- [ ] Launch referral program for merchant-to-merchant growth

**Target date**: 2026-06-30

## Goal 3: Improve Merchant Onboarding & Developer Experience

**Description**: Reduce friction in merchant onboarding and API integration so developers can go live faster

**Success criteria**:
- [ ] Launch developer portal with API docs, test credentials, and sandbox environment
- [ ] Reduce average integration time from 2 weeks to 3 days
- [ ] Publish SDK for top 3 platforms (Node, Python, PHP)

**Target date**: 2026-05-31
